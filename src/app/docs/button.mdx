# Button

Displays a button.

<Demo>
  <div className="w-fit mx-auto">
    <Button>Button</Button>
  </div>
</Demo>

```tsx
<Button />
```

### Outlined & Ghost

`Outlined` and `Ghost` variants inherit their colors from current text color.

<Demo>
  <div className="w-fit flex flex-wrap gap-3 mx-auto p-4 rounded-shape text-white bg-purple-500">
    <Button variant="outlined">Outlined</Button>
    <Button variant="ghost">Ghost</Button>
  </div>
</Demo>

```tsx
<div className="text-white bg-purple-500">
  <Button variant="outlined" />
  <Button variant="ghost" />
</div>
```
